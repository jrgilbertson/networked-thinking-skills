# Security

The latest release on `main` is the supported version.

These skills bundle helper scripts that run locally in your agent session, and remediation workflows can modify Obsidian vault files. Review audit output before allowing any vault mutation, and treat skill updates like any other dependency update.

To report a vulnerability in the scripts, schemas, or skill instructions, email [jason.gilbertson@gmail.com](mailto:jason.gilbertson@gmail.com) rather than opening a public issue. Non-sensitive bugs go to the [issue tracker](https://github.com/jrgilbertson/networked-thinking-skills/issues).
